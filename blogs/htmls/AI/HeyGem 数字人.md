## 产品介绍

HeyGem是一款开源的数字人解决方案，支持实时语音驱动和表情同步。

## 安装部署

### 系统要求
- Python 3.8+
- CUDA 11.7+
- 8GB+显存

### 安装步骤
```bash
pip install heygem
```

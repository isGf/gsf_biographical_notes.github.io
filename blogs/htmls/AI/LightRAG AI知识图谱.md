## 项目介绍

LightRAG是一个轻量级的检索增强生成（RAG）系统，专门用于构建和查询知识图谱。

## 核心特性

- 轻量级设计，易于部署
- 支持多种文档格式
- 智能知识图谱构建
- 高效的向量检索
- 可视化知识图谱

## 安装配置

### 环境要求
- Python 3.8+
- 支持OpenAI API或其他LLM服务

### 安装步骤
```bash
pip install lightrag
```
